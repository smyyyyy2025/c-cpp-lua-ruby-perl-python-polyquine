# c-cpp-lua-ruby-perl-python-polyquine

A polyglot quine that prints its own source code, valid in:

- C
- C++
- Lua
- Ruby
- Perl
- Python

## Run it

```bash
# C
gcc polyquine.c -o polyquine && ./polyquine

# C++
g++ polyquine.c -o polyquine && ./polyquine

# Python
python polyquine.c

# Lua
lua polyquine.c

# Ruby
ruby polyquine.c

# Perl
perl polyquine.c
